## Stopping an agreement

```bash
curl --request 'GET' \
     --url 'https://staging.payment.telenordigital.com/agreements/{agreementId}/stop' \
     --header 'content-type: application/json' \
     --header 'accept: application/json' \
     --user 'your-api-key'
```
